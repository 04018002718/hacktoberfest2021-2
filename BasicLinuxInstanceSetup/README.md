# Setting up your linux instance with basic developer tools
This script installs basic developer tools in your linux system
Made for debian based systems
This script installs:
a)g++
b)gdb
c)wget
d)vs-code installation
e)net-tools
f)openssh-server
g)pipenv

To run this file:
```chmod +x install.sh```
```sudo ./install.sh```
